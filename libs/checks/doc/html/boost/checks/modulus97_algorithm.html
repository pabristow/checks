Description

The mod97-10 algorithm (ISO/IEC 7064:2003 Information technology -- Security techniques -- Check character systems) uses two check digits.

This algorithm use two check digits.

  static checkdigits_iter

    Compute the two check digits with a simple modulus 97.

    Parameters:

    checkdigits

    is the output iterator in which the two check digits will be written.

    checksum

    is the checksum used to extract the check digit.

    Requires:

    checkdigits should have enough reserved place to store the two check digits.

    Postconditions:

    The two check digits are stored into checkdigits.

    Returns:

    An iterator initialized at one pass to the end of the two check digits.

    Throws:

    		 boost::checks::translation_exception if the check digits cannot be translated into the check digits_iter type.
