Description

strict_size_contract public static functions

  1. static bool reach_one_past_the_end(const size_t valid_value_counter);
    Tells if the expected interval of value [0..n) is outstripped.

    Parameters:

    valid_value_counter

    Number of valid values in the sequence already counted.

    Returns:

    true if valid_value_counter is one past the end of the expected size, else false.
