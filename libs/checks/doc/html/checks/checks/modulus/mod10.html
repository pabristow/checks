Modulus 10 algorithm

Description

This algorithm use a modulus 10 as the Luhn algorithm but use a custom weight pattern. The sum is made without subtraction if the multiplication of a digit exceeds 9. The custom weight pattern may be useful for many codes and numbers that aren't implemented in the high level library. The user can easily craft his own check function with this weight pattern.
