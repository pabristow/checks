Detailed Description

template<BOOST_PP_ENUM_BINARY_PARAMS(BOOST_CHECK_LIMIT_WEIGHTS, int weight_value,=0 BOOST_PP_INTERCEPT)>
class boost::checks::weight<>

The weight metafunction encapsulate 0 to BOOST_CHECK_LIMIT_WEIGHTS weights.

Template Parameters:
weight_valueThe weights are implicitly linked to a position through a sequence. No weight means an equal weight of 1 for all the digits.
Remarks:
There are BOOST_CHECK_LIMIT_WEIGHTS partial specialisations of this class.
